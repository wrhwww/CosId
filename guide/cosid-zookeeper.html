# CosId-ZooKeeper 模块

cosid-zookeeper (opens new window) 使用ZooKeeper作为雪花算法(SnowflakeId) 的 机器号分配器 (MachineIdDistributor) 、号段算法(SegmentId)的号段分发器 (IdSegmentDistributor)。

# 安装

Kotlin DSL

    implementation("me.ahoo.cosid:cosid-zookeeper:${cosidVersion}")

# ZookeeperIdSegmentDistributor

基于zookeeper的Id片段分发器。维护一个step步长id。

主要方法：获取下一个步长最大值

long nextMaxId(long step);

# ZookeeperMachineIdDistributor

基于zookeeper的机器号分发器。
